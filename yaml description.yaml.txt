API Name : OMDb API

description : The OMDb API is a RESTful web service to obtain movie information, all content and images on the site are contributed and maintained by our users.

API website link : https://www.omdbapi.com/

url for api provided on website : http://www.omdbapi.com/?apikey=[yourkey]&

url for api : "https://www.omdbapi.com/?apikey=2e6ace90&t={title}"
			(here title is the keyword we use to search data)

use of API : the api is used to search for movie details (shown below),details like movie title, imdb rating, poster, description, imdb id.


example search : https://www.omdbapi.com/?apikey=2e6ace90&t="The%20Avengers"
